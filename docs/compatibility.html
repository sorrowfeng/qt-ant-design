Compatibility

兼容性与质量

项目同时面向 Qt5 与 Qt6，重点处理控件默认样式、图标渲染、高 DPI、窗口系统和多分辨率显示差异。

Qt 版本 高 DPI 质量门禁 Pages 发布

Qt 版本

CMake 配置时使用 find_package(QT NAMES Qt6 Qt5 ...) 自动识别 Qt6 或 Qt5。当前重点验证环境包含 Qt 5.15.2 和 Qt 6.x。

  • Qt5 / Qt6 AntIcon 渲染保持同一 SVG 资源路径和视觉结果。
  • 输入框、树、列表、滚动区域、对话框和文件对话框覆盖专项视觉回归。
  • 安装包消费工程通过 find_package(qt-ant-design CONFIG REQUIRED) 验证。

高 DPI 与多分辨率

AntDesign::configureHighDpi() 在 Qt5 下启用逻辑高 DPI 缩放和高 DPI pixmap，并使用 pass-through scale rounding 策略减少 125%、150% 缩放下的尺寸漂移。

示例程序在主屏幕分辨率和缩放比例之外，还通过自动化测试覆盖 1.0、1.25、1.5 等比例的逻辑截图尺寸。

质量门禁

组件可靠性

公开组件覆盖 API、信号、生命周期、主题切换和渲染烟测。

视觉对齐

组件截图和 Qt5/Qt6 atlas 对比用于发现渲染差异。

无 QSS 示例

示例程序保持零样式表操作，避免绕开控件自身绘制。

真实页面遍历

示例程序会遍历全部页面并对滚动和空白渲染做检查。
